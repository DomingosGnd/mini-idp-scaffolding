# Manual de Utilização

## Criar novo projeto
1. Executar `python generate.py`
2. Informar nome, template e destino
3. Projeto será criado automaticamente


## Utilização via Web
1. Entrar na pasta `web`
2. Executar `python app.py`
3. Aceder a `http://localhost:5000`
4. Preencher o formulário para gerar projetos
